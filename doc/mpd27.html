4.8. IPv6CP layer

Mpd implements IPv6 Control Protocol (IPv6CP) for IPv6 link level addresses negotiation. To enable IPv6CP, ipv6cp option should be enabled at the bundle layer.

Now there is no additional configuration options present at this layer.
