2.1. Getting mpd

Mpd is supplied as a FreeBSD port, under the name net/mpd5. Therefore, the simplest way to get mpd onto your machine is to install the mpd package or port. For information on how to do this, see the FreeBSD ports web page.

Mpd can also be compiled and installed manually. This is useful for example if you want to change the set of device types supported by mpd.
